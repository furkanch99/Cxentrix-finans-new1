# Cxentrix Finans

Cxentrix Solutions için finans yönetim platformu.

## Kurulum

1. `npm install`
2. `.env` dosyası oluştur:
```
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key
```
3. `npm run dev`

## Yayın

Vercel'e otomatik deploy yapılır. Environment variable'lar Vercel dashboard'undan ayarlanmalı.
